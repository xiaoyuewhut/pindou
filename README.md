# 🫧 拼豆图纸生成器 · MARD221

> 上传图片 → 自动转为 MARD221 拼豆图纸 · 可编辑 · 可导出

一个纯前端的拼豆（Perler Beads）图纸生成工具，支持 221 色 MARD 配色方案，开箱即用，无需安装任何依赖。

## ✨ 功能特性

- **智能配色** — 基于 CIELAB 色彩空间的颜色匹配，比普通 RGB 欧氏距离更符合人眼感知
- **Floyd-Steinberg 抖动** — 可选的误差扩散算法，有效减少色带问题
- **丰富的编辑工具** — 画笔、吸管、橡皮、填充，支持撤销/重做（最多 60 步）
- **全局替换** — 一键将图纸中某个颜色全部替换为另一个颜色
- **多种导出格式**
  - 📄 带色号图纸 PNG — 高分辨率，每格标注色号
  - 🖼️ 位图 PNG — 纯色块，适合打印
  - 📊 材料清单 CSV — 含色号、名称、数量、占比
- **流畅交互** — 空格+拖拽平移画布、滚轮缩放、快捷键支持
- **221 色完整色卡** — 涵盖 A-H 系列 + M 系列（透明/夜光色）

```bash
# 或双击 index.html
```

## 📖 使用方法

1. **上传图片** — 点击上传区域或直接拖入图片（支持 JPG / PNG / WEBP）
2. **调整设置** — 设置图纸尺寸（默认 71×71）、是否启用抖动、是否允许透明色
3. **编辑图纸** — 使用左侧工具在画布上手动修改颜色
4. **导出** — 点击顶部按钮导出带色号图纸、位图或材料清单

## 🎨 关于 MARD221 配色

MARD221 是一种常用的拼豆配色方案，包含 221 种颜色：

| 系列 | 数量 | 说明 |
|------|------|------|
| A | 60 | 基础色 |
| B | 65 | 过渡色 |
| C | 12 | 荧光/夜光色 |
| D | 12 | 冷色系 |
| E | 12 | 暖色系 |
| F | 12 | 自然色系 |
| G | 10 | 深色系 |
| H | 23 | 高级灰/复古色 |
| M | 15 | 透明/夜光色 |

## 🛠️ 技术细节

- **颜色匹配算法** — sRGB → CIE XYZ → CIELAB 转换，使用 CIE76 距离计算最近色
- **性能优化** — RGB 量化缓存（步长 8），避免重复计算
- **导出安全** — Canvas 尺寸动态限制在 8192px 以内，防止内存溢出
